# Inventory Management System

## Overview
A full-featured inventory management system built with PHP, PostgreSQL, and Tailwind CSS. Designed for small to medium businesses.

## Features
- Secure authentication & role-based access
- Product, category, and supplier management (CRUD)
- Low stock alerts
- Activity logs (audit trail)
- Search & filtering
- Reports module

## Tech Stack
- PHP (PDO)
- PostgreSQL
- Tailwind CSS
- HTML5
- Secure password hashing
